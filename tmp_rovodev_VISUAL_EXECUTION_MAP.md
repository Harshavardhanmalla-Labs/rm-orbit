## 🎓 AGENT TIPS & TRICKS

```
💡 PRODUCTIVITY HACKS:

1. Start with the hardest task (eat the frog)
2. Use existing code as templates (don't reinvent)
3. Write tests FIRST (TDD saves time)
4. Commit frequently (small commits = easy rollback)
5. Take breaks (fresh eyes = fewer bugs)

🎯 QUALITY SHORTCUTS:

- Copy patterns from RM-Atlas (backend structure)
- Copy patterns from RM-Connect (frontend structure)
- Use shared UI components from Design System
- Run linter BEFORE committing (catch errors early)
- Test on mobile (responsive from day 1)

⚡ SPEED TIPS:

- Use AI code completion (GitHub Copilot)
- Keep documentation open in another tab
- Run tests in watch mode (instant feedback)
- Use hot reload (see changes immediately)
- Automate repetitive tasks (scripts FTW)
```
