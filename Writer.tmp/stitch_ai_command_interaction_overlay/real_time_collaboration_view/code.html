1. Overview

The new system architecture focuses on scalability and modularity. By decoupling the frontend from the backend services, we aim to improve deployment velocity and reduce regression risks. This document outlines the core components and their interactions.

1.1 Core Components

The primary components of the system include the API Gateway, the Authentication Service, and the Data Processing Pipeline. Each component is containerized and orchestrated via Kubernetes.
